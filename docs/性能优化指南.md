## 🔧 后端优化

### 1. 数据库连接池优化

**文件：** `backend/.env`

```bash
# 默认配置
DATABASE_POOL_MIN=1
DATABASE_POOL_MAX=10

# 生产环境配置
DATABASE_POOL_MIN=5
DATABASE_POOL_MAX=50
DATABASE_POOL_TIMEOUT=30
```

**代码优化：**
```rust
// db.rs
pub async fn create_pool() -> Result<PgPool, sqlx::Error> {
    let database_url = env::var("DATABASE_URL")
        .expect("DATABASE_URL must be set");
    
    PgPoolOptions::new()
        .min_connections(env::var("DATABASE_POOL_MIN")
            .ok()
            .and_then(|s| s.parse().ok())
            .unwrap_or(5))
        .max_connections(env::var("DATABASE_POOL_MAX")
            .ok()
            .and_then(|s| s.parse().ok())
            .unwrap_or(50))
        .acquire_timeout(Duration::from_secs(
            env::var("DATABASE_POOL_TIMEOUT")
                .ok()
                .and_then(|s| s.parse().ok())
                .unwrap_or(30)
        ))
        .connect(&database_url)
        .await
}
```

### 2. 查询优化

**添加数据库索引：**

```sql
-- 流量日志表索引
CREATE INDEX IF NOT EXISTS idx_traffic_logs_inbound_date 
ON traffic_logs(inbound_id, recorded_at DESC);

CREATE INDEX IF NOT EXISTS idx_traffic_logs_date 
ON traffic_logs(recorded_at DESC);

-- 入站配置表索引
CREATE INDEX IF NOT EXISTS idx_inbounds_user_enable 
ON inbound_configs(user_id, enable);

CREATE INDEX IF NOT EXISTS idx_inbounds_expire 
ON inbound_configs(expire_at) WHERE enable = true;

-- 用户表索引
CREATE INDEX IF NOT EXISTS idx_users_username 
ON users(username);

CREATE INDEX IF NOT EXISTS idx_users_email 
ON users(email) WHERE email IS NOT NULL;
```

**优化查询语句：**

```rust
// 优化前：N+1 查询
for inbound in inbounds {
    let user = sqlx::query("SELECT * FROM users WHERE id = $1")
        .bind(inbound.user_id)
        .fetch_one(&pool)
        .await?;
}

// 优化后：JOIN 查询
let results = sqlx::query_as::<_, (Inbound, Option<User>)>(
    r#"
    SELECT i.*, u.*
    FROM inbound_configs i
    LEFT JOIN users u ON i.user_id = u.id
    WHERE i.enable = true
    "#
)
.fetch_all(&pool)
.await?;
```

### 3. 缓存优化

**使用 Redis 缓存：**

```bash
# 添加 Redis 配置
REDIS_URL=redis://localhost:6379
CACHE_TTL=300  # 5 分钟
```

**缓存热点数据：**

```rust
use redis::AsyncCommands;

// 缓存系统状态
pub async fn get_cached_system_status(
    redis: &mut redis::aio::Connection,
    pool: &PgPool,
) -> Result<SystemStatus, Box<dyn Error>> {
    // 尝试从缓存获取
    let cached: Option<String> = redis.get("system_status").await?;
    if let Some(json) = cached {
        return Ok(serde_json::from_str(&json)?);
    }
    
    // 从数据库查询
    let status = query_system_status(pool).await?;
    
    // 写入缓存
    let json = serde_json::to_string(&status)?;
    redis::cmd("SET")
        .arg("system_status")
        .arg(&json)
        .arg("EX")
        .arg(300)  // 5 分钟 TTL
        .query_async(redis)
        .await?;
    
    Ok(status)
}
```

### 4. 异步并发优化

**使用 tokio 任务并发：**

```rust
use tokio::task::JoinSet;

// 并发处理多个入站配置
let mut tasks = JoinSet::new();

for inbound in inbounds {
    let pool = pool.clone();
    tasks.spawn(async move {
        process_inbound(&pool, inbound).await
    });
}

// 等待所有任务完成
while let Some(result) = tasks.join_next().await {
    result??;
}
```

## 🚀 Nginx 优化

### 生产环境配置

**文件：** `nginx.conf`

```nginx
worker_processes auto;
worker_rlimit_nofile 65535;

events {
    use epoll;
    worker_connections 4096;
    multi_accept on;
}

http {
    # 基础优化
    sendfile on;
    tcp_nopush on;
    tcp_nodelay on;
    keepalive_timeout 65;
    keepalive_requests 100;
    
    # Gzip 压缩
    gzip on;
    gzip_vary on;
    gzip_proxied any;
    gzip_comp_level 6;
    gzip_types text/plain text/css text/xml text/javascript
               application/json application/javascript application/xml
               application/xml+rss text/javascript application/x-javascript;
    
    # 缓存配置
    proxy_cache_path /var/cache/nginx levels=1:2
                     keys_zone=api_cache:10m max_size=1g
                     inactive=60m use_temp_path=off;
    
    # 后端代理
    upstream lx_proxy {
        server 127.0.0.1:8080;
        keepalive 32;
    }
    
    server {
        listen 80;
        server_name example.com;
        
        # 静态文件缓存
        location ~* \.(jpg|jpeg|png|gif|ico|css|js|webp)$ {
            expires 30d;
            add_header Cache-Control "public, immutable";
        }
        
        # API 代理
        location /api/ {
            proxy_pass http://lx_proxy;
            proxy_http_version 1.1;
            proxy_set_header Connection "";
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            
            # 缓存配置
            proxy_cache api_cache;
            proxy_cache_valid 200 1m;
            proxy_cache_valid 404 10s;
        }
    }
}
```
