## 🚀 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install vue-i18n@9
```

### 2. 配置 i18n

**文件：** `src/i18n/index.ts`

```typescript
import { createI18n } from 'vue-i18n'
import zhCN from './locales/zh-CN.json'
import enUS from './locales/en-US.json'
import zhTW from './locales/zh-TW.json'

const i18n = createI18n({
  legacy: false,
  locale: 'zh-CN', // 默认语言
  fallbackLocale: 'en-US',
  messages: {
    'zh-CN': zhCN,
    'en-US': enUS,
    'zh-TW': zhTW,
  },
})

export default i18n
```

### 3. 在 main.ts 中注册

```typescript
import { createApp } from 'vue'
import i18n from './i18n'

const app = createApp(App)
app.use(i18n)
app.mount('#app')
```

## 📝 使用方式

### 在组件中使用

#### Composition API (推荐)

```vue
<template>
  <div>
    <h1>{{ t('dashboard.title') }}</h1>
    <p>{{ t('common.loading') }}</p>
  </div>
</template>

<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t, locale } = useI18n()

// 切换语言
const changeLocale = (lang: string) => {
  locale.value = lang
}
</script>
```

#### Options API

```vue
<template>
  <div>
    <h1>{{ $t('dashboard.title') }}</h1>
  </div>
</template>

<script>
export default {
  methods: {
    changeLocale(lang) {
      this.$i18n.locale = lang
    }
  }
}
</script>
```

### 在 JavaScript 中使用

```typescript
import { useI18n } from 'vue-i18n'

const { t } = useI18n()

// 翻译文本
const message = t('common.success')

// 带参数的翻译
const daysLeft = t('time.daysLeft', { count: 5 })

// 复数形式
const users = t('users.total', { count: 10 })
```

```json

  "nav": { ... },           // 导航菜单
```

## 🎨 语言切换组件

### LocaleToggle 组件

```vue
<template>
  <el-dropdown trigger="click" @command="handleLocaleChange">
    <el-button circle>
      <span>{{ currentLocale.flag }}</span>
    </el-button>
    <template #dropdown>
      <el-dropdown-menu>
        <el-dropdown-item
          v-for="locale in supportedLocales"
          :key="locale.code"
          :command="locale.code"
        >
          {{ locale.flag }} {{ locale.name }}
        </el-dropdown-item>
      </el-dropdown-menu>
    </template>
  </el-dropdown>
</template>

<script setup lang="ts">
import { setLocale, getSupportedLocales } from '@/i18n'

const supportedLocales = getSupportedLocales()

const handleLocaleChange = (code: string) => {
  setLocale(code)
}
</script>
```

## 🔧 高级功能

### 1. 参数化翻译

```json
{
  "users": {
    "confirmDelete": "确定要删除用户 {username} 吗？"
  },
  "time": {
    "daysLeft": "剩余 {count} 天"
  }
}
```

```typescript
t('users.confirmDelete', { username: 'admin' })
// 输出：确定要删除用户 admin 吗？

t('time.daysLeft', { count: 5 })
// 输出：剩余 5 天
```

### 2. 复数形式

```json
{
  "users": {
    "total": "共 {count} 个用户",
    "total_0": "暂无用户",
    "total_1": "1 个用户"
  }
}
```

```typescript
t('users.total', { count: 0 })  // 暂无用户
t('users.total', { count: 1 })  // 1 个用户
t('users.total', { count: 10 }) // 共 10 个用户
```

### 3. 动态加载语言包

```typescript
// 按需加载语言包
async function loadLocale(locale: string) {
  const messages = await import(`./locales/${locale}.json`)
  i18n.global.setLocaleMessage(locale, messages.default)
}
```

### 4. 语言持久化

```typescript
// 保存到 localStorage
localStorage.setItem('locale', locale)

// 从 localStorage 读取
const saved = localStorage.getItem('locale')
```

## 📋 翻译指南

### 最佳实践

1. **使用描述性键名**
   ```
   ✅ good: 'users.createButton'
   ❌ bad: 'btn1', 'text1'
   ```

2. **避免硬编码文本**
   ```vue
   ✅ good: {{ t('common.submit') }}
   ❌ bad: 提交
   ```

3. **保持键名一致性**
   ```
   'users.title'
   'users.create'
   'users.edit'
   ```

4. **使用命名空间**
   ```
   module.entity.action
   users.list.delete
   ```

### 翻译注意事项

1. **上下文完整**
   - 提供足够的上下文信息
   - 避免歧义

2. **文化敏感性**
   - 避免文化特定的习语
   - 考虑不同地区的差异

3. **长度差异**
   - 英文通常比中文长 30-50%
   - 设计时预留足够空间

4. **特殊字符**
   - 注意引号、括号等字符
   - JSON 中需要转义

## 🌐 添加新语言

### 步骤

1. **创建语言包**
   ```bash
   cp src/i18n/locales/en-US.json src/i18n/locales/ja-JP.json
   ```

2. **翻译内容**
   ```json
   {
     "common": {
       "loading": "読み込み中...",
       ...
     }
   }
   ```

3. **注册语言**
   ```typescript
   import jaJP from './locales/ja-JP.json'
   
   const messages = {
     'zh-CN': zhCN,
     'en-US': enUS,
     'ja-JP': jaJP, // 新增
   }
   ```

4. **添加到语言列表**
   ```typescript
   export function getSupportedLocales() {
     return [
       { code: 'zh-CN', name: '简体中文', flag: '🇨🇳' },
       { code: 'en-US', name: 'English', flag: '🇺🇸' },
       { code: 'ja-JP', name: '日本語', flag: '🇯🇵' }, // 新增
     ]
   }
   ```
