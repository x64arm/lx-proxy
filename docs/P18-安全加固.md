## 📋 任务清单

### ✅ 1. SQL 注入防护审查

**当前状态：** ✅ 已防护

**防护措施：**
- 使用 SQLx 参数化查询（`$1`, `$2` 绑定参数）
- 所有用户输入通过 `.bind()` 传递
- 无字符串拼接 SQL

**检查点：**
- [x] 所有查询使用参数化
- [x] 动态表名/列名白名单验证
- [x] LIKE 查询使用参数化通配符

**验证方法：**
```bash
# 搜索字符串拼接 SQL
grep -r "format!(.*SELECT\|INSERT\|UPDATE\|DELETE" backend/src/
```

---

### ✅ 2. XSS 防护审查

**当前状态：** ✅ 已防护

**防护措施：**
- 前端使用 Vue 3 自动转义输出
- Element Plus 组件库内置 XSS 防护
- 禁用 `v-html` 或使用 DOMPurify 过滤

**检查点：**
- [x] Vue 模板无直接 HTML 渲染
- [x] 用户输入显示前自动转义
- [x] 订阅链接等 URL 使用 `target="_blank" rel="noopener"`

**需要补充：**
- [ ] 添加 DOMPurify 依赖（如必须使用 v-html）
- [ ] Content-Security-Policy 响应头

---

### ✅ 3. CSRF 防护

**当前状态：** ⚠️ 待实施

**防护方案：**
- JWT Token 存储在内存/LocalStorage
- 每次请求通过 `Authorization: Bearer <token>` 发送
- 浏览器不会自动发送 JWT，天然防 CSRF

**需要补充：**
- [ ] SameSite Cookie 属性（如使用 Cookie）
- [ ] 关键操作二次验证（密码/2FA）
- [ ] 添加 CSRF Token（可选，JWT 已足够）

---

### ✅ 4. 速率限制（防暴力破解）

**当前状态：** ✅ 已实现

**防护措施：**
- 登录接口：5 次/分钟，超限封禁 15 分钟
- API 接口：100 次/分钟，超限封禁 5 分钟
- 基于 IP 地址限流

**文件：**
- `backend/src/middleware/rate_limiter.rs`

**需要补充：**
- [ ] 基于用户 ID 限流（防止共享 IP）
- [ ] 分布式限流（Redis 实现）
- [ ] 限流监控告警

---

### ✅ 5. 敏感数据加密存储

- 密钥通过环境变量管理

**文件：**
- `backend/src/crypto/encryptor.rs`

**加密对象：**
- [x] SMTP 密码

### 3. 速率限制测试
```bash

for i in {1..10}; do
  curl -X POST http://localhost/api/auth/login \
    -d '{"username":"admin","password":"wrong"}'
done
# 第 6 次应该返回 429
```

### 4. HTTPS 测试
```bash
# 检查 HTTP 是否跳转 HTTPS
curl -I http://your-domain.com
# 应该返回 301 跳转到 HTTPS

# 检查 SSL 配置
nmap --script ssl-enum-ciphers -p 443 your-domain.com
```
