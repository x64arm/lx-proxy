## 🎨 核心技术特性

### 1. 响应式检测系统
```typescript
const isMobile = ref(false)
const viewMode = ref<'table' | 'card'>('table')

const checkDevice = () => {
  const width = window.innerWidth
  isMobile.value = width <= 768
  viewMode.value = isMobile.value ? 'card' : 'table'
}

onMounted(() => {
  checkDevice()
  window.addEventListener('resize', checkDevice)
})
```

### 2. 视图切换机制
- **自动切换：** 窗口宽度 ≤ 768px 自动切换
- **手动切换：** 移动端视图切换按钮

### 3. 触摸友好设计
```css
/* 最小点击区域 */
:deep(.el-button),
:deep(.el-menu-item),
:deep(.el-form-item__label) {
  min-height: 44px;
}

/* 移动端按钮 */
.mobile-actions .el-button {
  width: 100%;
  min-height: 48px;
}

/* 防止 iOS 自动缩放 */
.form-group input {
  font-size: 16px;
}
```

### 4. 卡片视图系统
```vue
<el-card class="user-card">
  <div class="card-header">
    <div class="card-id">#{{ user.id }}</div>
    <el-tag>{{ user.role }}</el-tag>
  </div>
  
  <div class="card-body">
    <div class="card-row">
      <span class="card-label">用户名：</span>
      <span class="card-value">{{ user.username }}</span>
    </div>
  </div>
  
  <div class="card-actions">
    <el-button type="primary">编辑</el-button>
    <el-button type="danger">删除</el-button>
  </div>
</el-card>
```

### 5. 表单适配方案
```vue
<!-- 响应式表单 -->
<el-form 
  :model="form" 
  label-width="auto" 
  :class="{ 'mobile-form': isMobile }"
>
  <el-button :block="isMobile">保存</el-button>
</el-form>
```

```css
.mobile-form :deep(.el-form-item__label) {
  width: 100% !important;
  margin-bottom: 8px;
}

.mobile-form :deep(.el-input) {
  width: 100% !important;
}
```
