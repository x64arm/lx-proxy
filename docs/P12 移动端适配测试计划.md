## 🚀 运行测试

### 自动化测试

#### 运行所有移动端测试
```bash
cd frontend
npx playwright test e2e/mobile-responsive.spec.ts
```

#### 运行特定设备测试
```bash
# iPhone 12
npx playwright test --project='Mobile Safari'

# Pixel 5
npx playwright test --project='Mobile Chrome'

# iPad
npx playwright test --project='iPad Air'
```

#### 使用测试脚本
```bash
./scripts/test-mobile.sh
```

#### 查看测试报告
```bash
npx playwright show-report test-results/report.html
```

### 真机测试

#### 步骤：
1. 在开发服务器启动后
2. 使用移动设备访问服务器 IP 地址
3. 按照测试用例逐项验证
4. 记录测试结果

#### 开发服务器启动：
```bash
cd frontend
npm run dev
```

## 🔧 调试技巧

### Playwright 调试
```bash
# 调试模式
npx playwright test --debug

# 追踪模式
npx playwright test --trace on

# 特定测试文件
npx playwright test e2e/mobile-responsive.spec.ts --debug
```

### 浏览器 DevTools
1. 在移动设备上打开 Chrome
2. 访问 `chrome://inspect`
3. 选择设备进行调试

### Safari Web Inspector
1. Mac 上打开 Safari 开发菜单
2. 连接 iOS 设备
3. 选择页面进行调试
