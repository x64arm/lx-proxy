## 🎯 任务清单

### 阶段一：API 文档自动化（OpenAPI/Swagger）[优先级：P0]
**预计时间：** 2-3 天

#### 任务分解
- [ ] **1.1 集成 utoipa crate**
  - 添加依赖到 Cargo.toml
  - 配置 OpenAPI 生成器
  
- [ ] **1.2 为 API 端点添加文档注解**
  - 认证模块（auth.rs）
  - 用户管理（handlers.rs）
  - 入站配置（handlers.rs）
  - 流量统计（handlers/traffic.rs）
  - 系统配置（handlers.rs）
  - TOTP 双因素（handlers/totp.rs）
  - 邮件通知（handlers/email.rs）
  - 节点管理（cluster/）
  - 审计日志（audit/）

- [ ] **1.3 创建 Swagger UI 页面**
  - 前端添加 swagger-ui 组件
  - 配置 API 文档路由
  - 添加文档访问权限控制

- [ ] **1.4 导出 OpenAPI 规范**
  - 生成 openapi.json
  - 生成 openapi.yaml
  - 添加到 docs/ 目录

**验收标准：**
- ✅ 访问 `/api/docs` 显示 Swagger UI
- ✅ 所有 API 端点有完整文档
- ✅ 支持在线测试 API
- ✅ 导出 OpenAPI 3.0 规范文件
