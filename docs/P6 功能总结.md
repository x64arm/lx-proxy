## 🚀 使用指南

### 国际化使用

#### 1. 安装依赖
```bash
cd frontend
npm install vue-i18n@9
```

#### 2. 注册 i18n
```typescript
// main.ts
import i18n from './i18n'
app.use(i18n)
```

#### 3. 在组件中使用
```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>

<template>
  <div>{{ t('dashboard.title') }}</div>
</template>
```

#### 4. 切换语言
```typescript
import { setLocale } from '@/i18n'
setLocale('en-US')
```

### 监控系统使用

#### 1. 启动监控服务
```bash
docker-compose -f docker-compose.yml \
  -f docker-compose.monitoring.yml \
  up -d
```

#### 2. 访问服务
- **Prometheus:** http://localhost:9090
- **Grafana:** http://localhost:3000
  - 用户名：admin
  - 密码：lx_grafana_admin

#### 3. 导入仪表板
1. 登录 Grafana
2. Dashboards → Import
3. 输入仪表板 ID 或上传 JSON
4. 选择 Prometheus 数据源

#### 4. 配置告警
1. Grafana → Alerting
2. 创建 Contact Point
3. 创建 Notification Policy
4. 为面板添加告警规则
