---

## ⚡ 快速开始（3 步）

### Step 1: 启动开发服务器
```bash
cd /root/.openclaw/workspace/lx-proxy/frontend
npm run dev
```

### Step 2: 运行测试
```bash
# 方式 A: 使用测试脚本（推荐）
cd /root/.openclaw/workspace/lx-proxy
./scripts/test-mobile.sh

# 方式 B: 直接运行
cd frontend
npx playwright test e2e/mobile-responsive.spec.ts
```

### Step 3: 查看报告
```bash
npx playwright show-report test-results/report.html
```

## 🎯 常用命令

### 运行所有测试
```bash
npx playwright test e2e/mobile-responsive.spec.ts
```

### 运行特定设备
```bash
# iPhone 12
npx playwright test --project='Mobile Safari'

# Android
npx playwright test --project='Mobile Chrome'

# iPad
npx playwright test --project='iPad Air'
```

### 调试模式
```bash
npx playwright test --debug
```

### 生成 HTML 报告
```bash
npx playwright test --reporter=html
npx playwright show-report
```

## ⚠️ 常见问题

### Q1: 测试超时
```bash
# 增加超时时间
npx playwright test --timeout=60000
```

### Q2: 浏览器未安装
```bash
npx playwright install --with-deps chromium
```

### Q3: 端口被占用
```bash
# 修改端口启动
npm run dev -- --port 5174
```
