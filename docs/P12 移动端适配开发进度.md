  - 移动端隐藏面包屑导航

  - 移动端隐藏用户名（只显示头像）
  - 触摸友好的点击区域（最小 44px）

- ✅ **响应式断点**
  ```css
  手机：≤ 768px
  平板：769px - 1024px
  桌面：> 1024px
  ```

- ✅ **主内容区优化**
  - 移动端 padding 减小（20px → 12px）
  - 自适应高度（calc(100vh - 60px)）
  - 防止水平滚动

#### 技术实现
```typescript
// 设备检测
const isMobile = ref(false)
const isTablet = ref(false)

const checkDevice = () => {
  const width = window.innerWidth
  isMobile.value = width <= 768
  isTablet.value = width > 768 && width <= 1024
}

// 监听窗口大小变化
window.addEventListener('resize', checkDevice)
```

## 🔧 技术方案

### 响应式断点标准
```css
:root {
  --breakpoint-xs: 375px;   /* 小手机 */
  --breakpoint-sm: 576px;   /* 大手机 */
  --breakpoint-md: 768px;   /* 平板 */
  --breakpoint-lg: 992px;   /* 小桌面 */
  --breakpoint-xl: 1200px;  /* 大桌面 */
}
```

### Element Plus 响应式栅格
```vue
<el-row :gutter="16">
  <el-col :xs="24" :sm="12" :md="8" :lg="6">
    <!-- 内容 -->
  </el-col>
</el-row>
```

### 移动端工具类
```css
/* 触摸友好的最小点击区域 */
.touch-target {
  min-width: 44px;
  min-height: 44px;
}

/* 移动端间距优化 */
@media (max-width: 768px) {
  .p-mobile-sm { padding: 8px !important; }
  .m-mobile-md { margin: 16px !important; }
}
```
