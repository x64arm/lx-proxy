

```bash

# Pixel 5 (Android)
npx playwright test --project='Mobile Chrome'

# iPad Air
npx playwright test --project='iPad Air'
```

## 🔍 常见问题排查

### 问题 1: 测试失败 "Timeout 30s"
**原因：** 开发服务器未启动或响应慢  
**解决：**
```bash
# 确保开发服务器运行
npm run dev

# 增加超时时间
npx playwright test --timeout=60000
```

### 问题 2: 浏览器未安装
**原因：** Playwright 浏览器未下载  
**解决：**
```bash
npx playwright install --with-deps chromium
```

### 问题 3: 端口冲突
**原因：** 5173 端口被占用  
**解决：**
```bash
# 修改 vite.config.ts 中的端口
# 或使用其他端口启动
npm run dev -- --port 5174
```

### 问题 4: 测试用例找不到元素
**原因：** 选择器路径变更  
**解决：**
```bash
# 使用调试模式
npx playwright test --debug

# 检查元素选择器是否正确
```
