## 🟡 待修复的编译问题

### 问题 1：Redis Error 转换
**位置：** `backend/src/cache.rs:99, 108`

**错误：**
```
the trait bound `RedisError: std::convert::From<(redis::ErrorKind, std::string::String)>` is not satisfied
```

**原因：** redis 0.27 版本不支持直接从元组创建 RedisError

**解决方案：** 需要升级到 redis 0.28+ 或使用其他错误处理方式

### 问题 2：WebSocket 路由状态类型
**位置：** `backend/src/main.rs:116, 151, 194`

**错误：**
```
expected `MethodRouter<Pool<Postgres>>`, found `MethodRouter<AppState>`
```

**原因：** WebSocket 路由需要 AppState，但 public_routes 使用的是 Pool

**解决方案：** 将 WebSocket 路由移到 protected_routes 或单独处理
