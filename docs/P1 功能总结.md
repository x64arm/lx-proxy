### 2️⃣ 邮件通知系统

**文件：** 
- `backend/src/email/mod.rs` - 邮件客户端
- `backend/src/email/templates.rs` - 邮件模板
- `backend/src/handlers/email.rs` - API 处理器

**功能特性：**
- ✅ SMTP 邮件发送支持
- ✅ 4 种精美 HTML 邮件模板
- ✅ 环境变量配置
- ✅ 错误处理和日志记录

**邮件模板类型：**

1. **流量告警邮件** (`traffic_alert`)
   - 触发条件：流量使用达 70%/90%
   - 显示进度条和百分比
   - 颜色编码（绿/黄/红）

2. **到期提醒邮件** (`expiry_reminder`)
   - 触发条件：配置即将到期
   - 显示剩余天数
   - 紧急程度分类（提醒/重要/紧急）

3. **禁用通知邮件** (`disabled_notification`)
   - 触发条件：配置被禁用
   - 说明禁用原因（过期/超限）
   - 提供联系管理员提示

4. **测试邮件** (`test_email`)
   - 用于验证 SMTP 配置
   - 成功/失败反馈

**API 端点：**
```bash
# 发送测试邮件
POST /api/email/test
Content-Type: application/json

{
  "email": "user@example.com"
}

# 获取邮件配置状态
GET /api/email/status

# 响应示例
{
  "configured": true,
  "smtp_server": "smtp.gmail.com",
  "smtp_port": 587,
  "from_email": "noreply@lx-proxy.com"
}
```

## 🔧 配置说明

### 环境变量配置

**1. Xray API 配置**
```bash
XRAY_API_URL=http://127.0.0.1:62780
```

**2. SMTP 邮件配置**
```bash
# 必填
SMTP_SERVER=smtp.gmail.com
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=your-app-password

# 可选（有默认值）
SMTP_PORT=587
SMTP_FROM_EMAIL=your-email@gmail.com
SMTP_FROM_NAME=LX-Proxy
```

**3. Gmail SMTP 配置示例**
```bash
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=abcd efgh ijkl mnop  # 应用专用密码
SMTP_FROM_EMAIL=your-email@gmail.com
SMTP_FROM_NAME=LX-Proxy
```

**4. Outlook SMTP 配置示例**
```bash
SMTP_SERVER=smtp.office365.com
SMTP_PORT=587
SMTP_USERNAME=your-email@outlook.com
SMTP_PASSWORD=your-password
```

## 📝 注意事项

1. **邮件发送失败处理**
   - 检查 SMTP 配置是否正确
   - 确认防火墙允许 SMTP 端口（587/465）
   - Gmail 用户需使用应用专用密码
   - 查看后端日志获取详细错误信息

2. **Xray API 连接失败**
   - 确认 Xray-core 已启动并启用 API
   - 检查 `XRAY_API_URL` 配置
   - 确认端口 62780 未被占用

3. **定时任务未执行**
   - 检查后端日志确认任务已启动
   - 确认服务器时间正确
   - 查看任务执行日志
