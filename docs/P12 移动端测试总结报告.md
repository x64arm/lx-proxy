## 🔍 技术问题分析

### 问题 1：登录流程超时
**现象：** 登录后等待 URL 跳转超时 30 秒

**可能原因：**
- 前端路由配置与测试不匹配
- 登录 API 未实现或失败
- 认证逻辑问题

**建议修复：**
```typescript
// 使用更宽松的等待条件
await page.waitForURL(/\/(dashboard|users|inbounds|#)/, { 
  timeout: 10000,
  waitUntil: 'domcontentloaded'
});
```

### 问题 2：浏览器未安装
**现象：** Firefox/WebKit 测试立即失败（2ms）

**解决方案：**
```bash
npx playwright install firefox webkit
```

### 问题 3：选择器不匹配
**现象：** `.login-box` 元素找不到

**可能原因：**
- Vue 组件未完全渲染
- 类名与实际不符
- 需要等待 DOM 更新

**建议修复：**
```typescript
// 等待 Vue 组件渲染
await page.waitForSelector('#app', { state: 'attached' });
await expect(page.locator('.login-container')).toBeVisible();
```
